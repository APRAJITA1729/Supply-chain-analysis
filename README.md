# Supply-chain-analysis
Supply Chain Delivery Performance Analysis project using Python, Data Analytics, Visualization, and Machine Learning. This project analyzes delivery delays, identifies operational bottlenecks, performs profitability analysis, and predicts late deliveries using a Random Forest Classifier.
## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- SMOTE

## Key Results
- 54.71% orders delivered late
- $2.1M profit at risk
- Random Forest Accuracy: 74%
